Formatação de textos

Texto em negrito, com a tag semântica <strong> texto </strong>
Texto em itálico, com a tag a semântica <em> texto </em>
Texto marcado, com a tag semântica <mark> texto </mark> é tipo uma caneta marca texto.
Texto grande, com a tag obsoleta <big> texto </big>
Texto pequeno, com a tag semântica <small> texto </small>
Texto deletado, com a tag obsoleta <strike> texto </strike>
Texto inserido, com a tag <ins> texto </ins>
Textos sobrescrito e subscrito, x2 e H2O, com as tags <sub> texto </sub>, <sup> texto </sup>
